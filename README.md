##Projeto: Analisando dados de um Dashboard de Vendas no Power BI

Na atividade, eu repliquei as duas páginas do relatório apresentadas no curso usando os dados disponibilizados pela instrutora. Depois, criei uma terceira página original no Power BI Desktop para treinar a construção de visuais. Nessa página, incluí dois mapas: um mostrando a soma de vendas e unidades vendidas por país e outro exibindo a soma do lucro por país. Também adicionei um gráfico de pizza para apresentar o lucro por segmento.
